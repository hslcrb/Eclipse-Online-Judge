**5가지 고양이 포즈 (각 4초):**

1. **Sit (앉기)** 🪑
   - 앞다리 세우고 뒷다리 구부림
   - 머리 끄덕끄덕

2. **Stretch (기지개)** 🙆
   - 몸을 앞으로 기울임
   - 앞발을 쭉 뻗음
   - 꼬리 치켜세움

3. **Lay Down (눕기)** 😴
   - 몸을 낮춤
   - 앞발 쭉 펴기
   - 편안한 자세

4. **Playful (장난치기)** 🎾
   - 엉덩이 들기 (사냥 자세)
   - 통통 튀기
   - 꼬리 미친듯이 흔들기

5. **Groom (그루밍)** 🧼
   - 왼발 들어 세수하기
   - 머리 돌리기
   - 고양이 특유의 세수 동작

**항상 작동하는 애니메이션:**
- 귀 미세한 움직임
- 꼬리 자연스러운 흔들림
- 눈동자 미세한 떨림

### ✅ 입력 시스템 정비 확인

**Line Match Mode:**
- ✅ **정확한 매칭**: `s.text === input`
- ✅ **공백 포함**: 들여쓰기, 띄어쓰기 모두 정확히
- ✅ **대소문자 구분**: 정확히 일치해야 함
- ✅ **특수문자 포함**: 괄호, 세미콜론 등 모두

**예시:**
```javascript
// 스니펫
"if (document.frm.custno.value.length == 0) {"

// 정답 ✅
"if (document.frm.custno.value.length == 0) {"

// 오답 ❌
"false ffd gfg"  // 완전히 다름
"if(document..."  // 공백 누락
```

**Typing Rain Mode:**
- ✅ **실시간 매칭**: 입력 중 떨어지는 단어와 비교
- ✅ **황금색 표시**: 매칭되면 즉시 피드백
- ✅ **자유 입력 허용**: 매칭 안 되어도 +20점
- ✅ **정확한 비교**: `w.word === value`

### 🔧 동작 방식

**Line Match:**
1. 화면에 떨어지는 스니펫 확인
2. **정확히 똑같이** 입력 (공백, 대소문자 포함)
3. Enter로 제출
4. 일치하면 점수, 불일치하면 감점

**Typing Rain:**
1. 떨어지는 단어 확인
2. 입력하면 실시간 매칭
3. 매칭되면 황금색 표시
4. Enter로 캡처
