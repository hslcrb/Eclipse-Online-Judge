## 실시간 채점 시스템 완성:

### 1. **Ctrl+S 저장** 💾
- `Ctrl+S` (Mac: `Cmd+S`) 누르면 저장
- 화면 중앙에 "💾 저장 완료!" 알림
- 2초간 표시 후 자동 사라짐

### 2. **실시간 채점 알고리즘** 🎯

#### **정확도 계산**
```
정확도 = (올바른 문자 수 / 전체 입력 문자 수) × 100
```
- 문자 단위로 실시간 비교
- 타이핑 중 즉시 업데이트

#### **실수 카운터**
- 잘못 입력한 문자 개수 추적
- 빨간색 ❌로 표시

#### **점수 계산 공식**
```
라인 점수 = 100 × 콤보배율 × (1 + 정확도보너스 × 0.1)
정확도보너스 = floor(정확도 / 10)
```

#### **최종 보너스**
```
완성 보너스 = 시간보너스 + 정확도보너스 - 실수패널티 + 5000
- 시간보너스: max(0, (300 - 소요시간) × 10)
- 정확도보너스: 정확도 × 50
- 실수패널티: 실수횟수 × 10
```

### 3. **HUD 표시** 📊
```
⭐ 점수
📝 완성줄/전체줄
🎯 정확도% (색상 코딩)
   - 95% 이상: 초록색
   - 80~94%: 노란색
   - 80% 미만: 빨간색
❌ 실수 (0이면 표시 안 함)
🔥 COMBO (5콤보 이상)
📄 파일명
💡 Ctrl+S 저장
```

### 4. **완성 시 상세 통계** 📈
