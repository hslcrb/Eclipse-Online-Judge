## 🐸 개구리 펫 개선 사항

### 1. **재질 업그레이드**
```typescript
// 이전: Phong Material
MeshPhongMaterial({ 
  color: 0x7FFF00,
  shininess: 60,
  specular: 0x224422
})

// 현재: Standard Material (PBR)
MeshStandardMaterial({ 
  color: 0x7FFF00,
  roughness: 0.3,      // 표면 거칠기
  metalness: 0.1,      // 금속성
  emissive: 0x001100,  // 자체 발광
  emissiveIntensity: 0.05
})
```

**효과**: 
- 더 사실적인 빛 반사
- 촉촉한 개구리 피부 느낌
- 환경 조명과 자연스러운 상호작용

### 2. **Geometry 해상도 대폭 증가**
```typescript
// 몸통/머리
SphereGeometry(radius, 32, 32)  // 이전
SphereGeometry(radius, 64, 64)  // 현재 (+100%)

// 눈 베이스
SphereGeometry(0.3, 20, 20)     // 이전
SphereGeometry(0.32, 48, 48)    // 현재 (+140%)

// 팔다리
CylinderGeometry(r, r, h, 16)   // 이전
CylinderGeometry(r, r, h, 24-32) // 현재 (+50~100%)
```

**효과**: 
- 훨씬 부드러운 표면
- 각진 느낌 완전 제거
- 빛 반사가 더 자연스러움

### 3. **눈 시스템 개선**

#### 이전
- 눈 베이스 (0.3 크기, 20 segments)
- 동공 (0.11 크기)
- 하이라이트 1개

#### 현재
- **눈 베이스** (0.32 크기, 48 segments, 타원형 scaling)
- **더 큰 동공** (0.13 크기, 32 segments)
- **하이라이트 2개** (주 하이라이트 + 보조 하이라이트)
  - 주: 0.05 크기, 100% 불투명
  - 보조: 0.03 크기, 70% 투명
- **receiveShadow 활성화**

**효과**: 
- 눈이 훨씬 생동감 있음
- 입체감 증가
- 반짝이는 느낌

### 4. **입 구조 정교화**

#### 이전
- 단순한 토러스 1개

#### 현재
- **윗입술**: 토러스 (r=0.18, 16 segments)
- **아랫입술**: 토러스 (r=0.15, 약간 작음)
- **입 가장자리**: 구체 2개 (좌우)
- **자연스러운 미소 곡선**

**효과**: 
- 입이 더 귀엽고 표정이 풍부
- 3D 입체감

### 5. **손 시스템 개선**

#### 이전
```typescript
// 손: 단순한 구체
// 손가락 3개: 단순한 원통
```

#### 현재
```typescript
// 손: 타원형 구체 (1.25 × 0.85 × 1.05 scale)
// 손가락 4개: 각각 독립적 위치
fingerPositions = [
  { x: -1.30, y: 1.22, z: 0.12, rot: -0.6 },
  { x: -1.35, y: 1.20, z: 0.04, rot: -0.5 },
  { x: -1.38, y: 1.18, z: -0.04, rot: -0.4 },
  { x: -1.35, y: 1.16, z: -0.12, rot: -0.3 },
]
// 각 손가락 끝에 둥근 구체 추가
```

**효과**: 
- 손가락이 훨씬 자연스러움
- 개별 손가락 인식 가능
- 손끝 둥글게 처리

### 6. **발 시스템 개선**

#### 이전
```typescript
// 발가락 5개: 단순한 원통 (균일한 크기)
for (let i = 0; i < 5; i++) {
  // 일정한 간격, 일정한 크기
}
```

#### 현재
```typescript
// 발가락 5개: 크기가 다른 자연스러운 배치
toePositions = [
  { x: -0.50, z: 0.58, size: 0.045 }, // 새끼발가락 (작음)
  { x: -0.43, z: 0.62, size: 0.050 },
  { x: -0.38, z: 0.64, size: 0.052 }, // 가운데 (제일 큼)
  { x: -0.33, z: 0.62, size: 0.050 },
  { x: -0.26, z: 0.58, size: 0.045 }, // 엄지 (작음)
]
// 각 발가락 끝에 둥근 구체
```

**효과**: 
- 실제 발처럼 가운데가 가장 김
- 발가락 끝이 둥글게 처리
- 훨씬 자연스러운 배치

### 7. **관절 추가**

#### 추가된 관절
- **팔꿈치 관절**: 구체 (r=0.15, 20 segments)
- **발목 관절**: 구체 (r=0.18, 20 segments)

**효과**: 
- 팔다리가 자연스럽게 연결
- 움직임이 더 사실적

### 8. **다리 강화**

#### 이전
```typescript

```

#### 현재
```typescript

kneeGeometry(0.26, 24, 24)            // 더 큰 무릎
calfGeometry(0.25, 0.20, 0.55, 32)    // 더 자연스러운 곡선
```

**효과**: 
- 튼튼하고 근육질인 개구리 다리
- 점프할 것 같은 느낌

### 9. **등 무늬 개선**

#### 이전
```typescript

```

#### 현재
```typescript

{ x: -0.32, y: 2.05, z: -0.52, size: 0.13, opacity: 0.8 }
{ x: 0.22, y: 2.15, z: -0.48, size: 0.11, opacity: 0.75 }
// ... 각각 다른 투명도
```

**효과**: 
- 무늬가 더 자연스럽게 피부와 어우러짐
- 깊이감 증가

### 10. **그림자 시스템**

#### 추가된 기능
```typescript

```

**적용 부위**: 
- 몸통, 머리, 눈 베이스
- 모든 팔다리 관절
- 손, 발

### 6. **수염 시스템 개선**

#### 이전
```typescript

  for (let i = 0; i < 3; i++) {
    points = [start, end];
  }

```

#### 현재
```typescript
// 곡선 8개 (CatmullRom Curve)
leftWhiskers = [
  { start, end, curve: 0.02 },  // 위로 휘어짐
  { start, end, curve: 0.01 },
  { start, end, curve: 0 },     // 직선
  { start, end, curve: -0.01 }, // 아래로 휘어짐
]

// 3점 곡선
points = [start, midpoint, end]
curve = new THREE.CatmullRomCurve3(points)
geometry.setFromPoints(curve.getPoints(20))
```

**효과**: 
- 자연스럽게 휘어진 수염
- 실제 고양이처럼

## 🎨 공통 개선 사항

### 1. **PBR 재질로 업그레이드**
- **MeshPhongMaterial** → **MeshStandardMaterial**
- 더 사실적인 빛 반사
- roughness, metalness 파라미터 활용

### 2. **Geometry 해상도 대폭 증가**
- 평균 **+50~100%** segments 증가
- 부드러운 곡선
- 각진 느낌 완전 제거

### 3. **그림자 시스템 추가**
- `castShadow`: 그림자 생성
- `receiveShadow`: 그림자 받기
- 더 입체적인 느낌

### 4. **디테일 추가**
- 개구리: 팔꿈치/발목 관절, 손발가락 끝 둥글게
- 고양이: 발 패드, 귀 털, Y자 입선

### 5. **투명도 활용**
- 무늬에 투명도 추가
- 하이라이트에 부분 투명도
- 더 자연스러운 표현
