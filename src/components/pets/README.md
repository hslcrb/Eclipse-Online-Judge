# 🐸 Pet System

완전히 독립적인 펫 시스템 - 모든 레이어 위에 떠다니는 귀여운 펫들!

## 구현된 펫

### 1. FrogPet (이족보행 개구리)

**특징:**
- ✅ Three.js 3D 렌더링
- ✅ 투명 배경 (alpha: true)
- ✅ 최상위 레이어 (z-index: 99999)
- ✅ 완전히 독립적인 컴포넌트
- ✅ 드래그 가능 (화면 어디든 이동)
- ✅ 커스텀 우클릭 메뉴 (펫 전용 격리)
- ✅ 우클릭으로 끄기 기능

**우아한 춤 동작 (5가지 페이즈):**

1. **오른팔 내밀기 (3초)**
   - 오른팔을 천천히 앞으로 내밈
   - 허리를 오른쪽으로 축축 흔들기
   - 머리도 함께 살짝 움직임

2. **왼팔 내밀기 (3초)**
   - 왼팔을 천천히 앞으로 내밈
   - 허리를 왼쪽으로 축축 흔들기
   - 리듬감 있는 동작

3. **오른발 내밀기 (3초)**
   - 오른발을 앞으로 내밈
   - 허리를 돌리며 춤
   - 머리를 끄덕끄덕

4. **왼발 내밀기 (3초)**
   - 왼발을 앞으로 내밈
   - 반대 방향으로 허리 돌리기
   - 균형잡힌 동작

5. **벽타기 시뮬레이션 (3초)**
   - 몸을 앞으로 기울임
   - 팔다리를 교차로 움직임 (기어오르는 모션)
   - 위아래로 살짝 움직임 (등반 느낌)
   - 머리를 위로 향함

**구조:**
```
개구리
├── 머리 (큰 구체)
│   ├── 왼쪽 눈 (튀어나온 눈)
│   ├── 오른쪽 눈
│   └── 입
├── 몸통 (타원형)
│   └── 배 (노란색 부분)
├── 왼팔 (원통형)
│   └── 왼손 (구체)
├── 오른팔
│   └── 오른손
├── 왼다리 (두꺼운 원통)
│   └── 왼발 (박스)
├── 오른다리
│   └── 오른발
└── 등 무늬 (5개 반점)
```

**색상:**
- 메인: `#7FFF00` (밝은 녹색)
- 다크: `#228B22` (진한 녹색)
- 배: `#FFFF99` (노란색)

**인터랙션:**
- **좌클릭 + 드래그**: 개구리 이동
- **우클릭**: 커스텀 메뉴 열기
  - 🐸 개구리 끄기 (빨간색 옵션)
  - 도움말 표시

**커스텀 우클릭 메뉴:**
```typescript
// 펫 컴포넌트에만 격리된 독립 메뉴
- 다크 테마 스타일 (#2B2B2B 배경)
- 호버 효과 (빨간색 하이라이트)
- 부드러운 그림자
- 클릭 외부 영역 시 자동 닫힘
- 다른 UI와 완전 독립
```

**사용법:**
```tsx
import FrogPet from '@/components/pets/FrogPet';

<FrogPet />
```

**애니메이션 타이밍:**
- 전체 사이클: 15초 (5개 페이즈 × 3초)
- 부드러운 전환: sine 함수 기반 보간
- 천천히 우아하게 움직임 (time += 0.02)
